<h1 align="center">Super Store Sales Analysis</h1>

<p align="center">
An interactive Power BI dashboard project analyzing Super Store sales data to uncover insights about revenue, profit, categories, regions, and customer segments using Excel and Power BI.
</p>

<hr>

<h2>Project Overview</h2>

<p>
This project, <strong>Super Store Sales Analysis</strong>, provides a complete business performance overview of a retail Super Store. It uses Excel for data preparation and Power BI for visualization to analyze key metrics such as total sales, profit, order trends, category performance, and regional insights — enabling data-driven decision-making.
</p>

<hr>

<hr>

<h2>Key Features</h2>
<ul>
  <li>Total sales, profit, and order analysis</li>
  <li>Sales and profit breakdown by category and sub-category</li>
  <li>Regional and state-wise performance analysis</li>
  <li>Customer segment insights (Consumer, Corporate, Home Office)</li>
  <li>Monthly and yearly sales trends</li>
  <li>Top-performing and underperforming products</li>
  <li>Interactive filters and dynamic visuals</li>
</ul>

<hr>

<h2>Technologies Used</h2>
<ul>
  <li>Microsoft Excel – for data cleaning and preparation</li>
  <li>Power BI – for building interactive dashboards</li>
</ul>

<hr>

<h2>How to Use</h2>
<ol>
  <li>Open the <code>Sales Data.csv</code> file in Excel to explore raw data.</li>
  <li>Open the <code>Superstore Sales Project.pbix</code> file in Power BI Desktop.</li>
  <li>Use slicers and filters to explore insights interactively.</li>
</ol>
